### `Performance Monitor`

通过`Performance`只能调试页面初次加载或者手动记录某个情形的性能瓶颈，但是通过`performance monitor`可以实时的查看各种性能数据。

```shell
command + shift + p

# 然后输入“Show Performance Monitor”
```
