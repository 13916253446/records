### 注释给使用者带来提示

无论是`ts`还是`js`申明变量都可以采用`jsdoc`注释的形式，给别的同事使用的时候带来提示。

```javascript
/**
 * 测试方法
 * @param  {} name 名字
 * @returns String 返回一个名字
 */
function test (name) {
  return name
}
```
